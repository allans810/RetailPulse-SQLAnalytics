RetailPulse Analytics Project

RetailPulse is a SQL-based e-commerce analytics project that models a real online retail business. It includes a full relational database schema, large synthetic datasets, and SQL solutions to real business problems.


Project Structure

RetailPulse-Analytics/
│
├── sql/
│   ├── schema.sql
│   └── BusinessRequests.sql
│
├── data/
│   ├── customers.csv
│   ├── products.csv
│   ├── orders.csv
│   ├── order_items.csv
│   └── payments.csv
│
├── visuals/
│   └── erd.png
│
├── docs/
│   └── Business_Requests.word
│
└── README.md


Database Schema

The database includes five core tables: customers, products, orders, order_items, payments.


What This Project Includes

* Complete SQL schema design
* Realistic sample datasets (2024–2025)
* Business queries covering revenue, customers, products, cities, payments, and rankings
* Clean ERD visualization
* Insights and documentation


Skills Demonstrated

SQL (Joins, Aggregations, Window Functions, CTEs, Date Functions)
Database design & ERD modeling
Business analytics & insight generation

 
How to Use

1. Import schema.sql
2. Load CSVs from `/data`
3. Run business queries in BusinessRequests.sql
